# Contribuir

1. Toda hipótesis debe incluir una predicción nula explícita.
2. Distinguir protocolo confirmatorio de análisis exploratorio.
3. No describir correlación como causalidad ni Bell como señalización.
4. Añadir fuente primaria para afirmaciones experimentales.
5. Incluir simulación o control positivo cuando sea posible.
6. Los cambios de análisis crítico requieren datos sintéticos y tests.
